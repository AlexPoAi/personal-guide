# Персональное руководство

Это твоё личное пространство для развития. Здесь собраны ориентиры, практики и планы — не как обязаловка, а как живой документ который ты обновляешь вместе с ИИ.

## Что здесь есть

- **profile.md** — твой текущий уровень и на чём сосредоточиться
- **worldview.md** — ключевая идея которая помогает двигаться
- **methods.md** — конкретные практики на эту неделю
- **weekly/** — планы по неделям
- **daily/** — что делать сегодня

## Как пользоваться

Открывай `daily/` каждое утро. Раз в неделю смотри `weekly/`. Когда что-то меняется в жизни или работе — обновляй `profile.md` вместе с ИИ.
